Train Scheduler

A responsive Javascript application that monitors train schedules and calculates real-time train arrival data using the Moment.js library. Incorporates Firebase Authentication and Database services.

LIRI

This Language Interpretation and Recognition Interface is a Node.js CLI application that allows a user to query the Twitter, Spotify and OMDB API's.

Flashcard Generator

This Node.js CLI application allows a user to generate two different types of flashcard decks for the purposing of studying. Each new card added to a deck is created using a Javascript Constructor. The user can then practice with either type of deck.

Bamazon

An Amazon.com-like Node.js CLI app built with the Inquirer NPM Library. Store inventory data is maintained within a MySQL database.

Friend Finder

A Responsive, full-stack app that matches compatible friends by evaluating responses from a personality survey. Back-end is built on Node.js and the Express Framework, using Handlebars.js as a view engine.

Burger

MySQL Edition

A responsive, full-stack app that allows users to track which hamburgers around town they have eaten. This MVC-designed app is built on Node.js with Express Routing, a MySQL database, a homemade ORM and the Handlebars.js view engine.

Burger

Sequelize Edition

A responsive, full-stack app that allows users to create a list of burgers that they wish to eat and then track when they finally devoured each one. This MVC-designed app is built with Node.js and the Express Framework, a MySQL database and the Handlebars.js view engine.
