Most of the applications below were weekly exercises built while attending the UT Coding Bootcamp. Arranged from my most recent projects to the earliest experiments, this chronology illustrates the progress in my programming comprehension and ability: graduating from simple front-end apps with vanilla html, css and javascript to full-stack apps with a Node.js Express server, Mongo Database and React UI.

Grocery List

Wireframe of full-stack app currently in development. Build includes Node, Express, React, Redux, PassportJS, MongoDB and Ant Design UI. App will generate an aggregate, ingredient-sorted weekly grocery list compiled from a cache of user-selected recipes in accordance with user-specified preferences and dietary restrictions.

Emailer

Full-stack application built in a Stephen Grider course. Node-Express server, Mongo Database, React front-end, & Redux state management. Authentication via Google OAuth, PassportJS and cookies. Credit card payments via Stripe API. Email dispatch via SendGrid API. Front-end styling with styled-components & react-materialize libraries.

Javascript Subreddit

webscraper

A Node.js application that scrapes articles from the Javascript subreddit. Users can save articles and then leave comments. Articles and comments are stored in MongoDB via mLab. Uses Mongoose, Handlebars and Cheerio NPM libraries.

Burger

Sequelize Edition

A responsive, full-stack app that allows users to create a list of burgers that they wish to eat and then track when they finally devoured each one. This MVC-designed app is built with Node.js and the Express Framework, a MySQL database with a Sequelize ORM and the Handlebars.js view engine. Vex Library for Modal dialogue boxes.

Burger

MySQL Edition

A responsive, full-stack app that allows users to track which hamburgers around town they have eaten. This MVC-designed app is built on Node.js with Express Routing, a MySQL database, a homemade ORM and the Handlebars.js view engine.

Friend Finder

A Responsive, full-stack app that matches compatible friends by evaluating responses from a personality survey. Back-end is built on Node.js and the Express Framework, using Handlebars.js as a view engine.

Bamazon

An Amazon.com-like Node.js CLI app built with the Inquirer NPM Library. Store inventory data is maintained within a MySQL database.

Flashcard Generator

This Node.js CLI application allows a user to generate two different types of flashcard decks for the purposing of studying. Each new card added to a deck is created using a Javascript Constructor. The user can then practice with either type of deck.

LIRI

This Language Interpretation and Recognition Interface is a Node.js CLI application that allows a user to query the Twitter, Spotify and OMDB API's.

Train Scheduler

A RESPONSIVE JAVASCRIPT APPLICATION THAT MONITORS TRAIN SCHEDULES AND CALCULATES REAL-TIME TRAIN ARRIVAL DATA USING THE MOMENT.JS LIBRARY. INCORPORATES FIREBASE AUTHENTICATION AND DATABASE SERVICES.

Trivia Game

My SECOND application! A responsive Javascript application that focuses on the use of Javascript Timing Events.

Hangman

My FIRST application! A simple version of the classic Hangman game. Built with Javascript and jQuery.

About

I am a freshly minted full-stack javascript web developer. I arrived here after reflecting upon the trajectory of my life’s path, having accrued extensive knowledge via various academic institutions, gained perspective and wisdom through diverse career experiences and developed fascinations with numerous technology-rich pastimes. A single theme bubbled to the surface as a result of this introspection: the joy of comprehending the design and operation of technology systems. With a background steeped in information analysis, diagnostic thinking and the detail-oriented operation of various technologies, transitioning my intellectual focus to the realm of programming languages is a natural and timely step forward upon an exciting new path. I'm convinced that a challenging career in software development will be both personally fulfilling and of positive benefit to the world, and this empowers my decision to strive towards mastery of this new skill set.

The delight of this earthly sojourn would be immeasurably diminished were it not for my love, the exceptionally talented and inspirational Alyssa Lytle. Daily we put pen to paper, composing a seldom known Craigslist success story.
